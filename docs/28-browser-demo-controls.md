# Build a Browser Demonstration Controller

Add development controls:

```text
W / Up Arrow: walk away from camera
S / Down Arrow: walk toward camera
A / Left Arrow: walk left
D / Right Arrow: walk right

Q: rotate left
E: rotate right

1: neutral
2: happy
3: thinking
4: surprised
5: skeptical

I: idle
P: point
X: explain
C: cast
R: react

Space: stop
Home: return to center
O: walk clockwise circle
Shift+O: walk counterclockwise circle
```

The browser must send semantic commands to the server.

Do not move the character only on the client.

The server remains authoritative.
