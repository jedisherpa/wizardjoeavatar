# Implement Blinking

Use:

```text
open
half_closed
closed
half_closed
open
```

Timing:

```text
total blink length: 120–180 ms
ordinary interval: 3–7 seconds
occasional double blink: allowed
```

Use a deterministic seeded random generator for tests.

Blink only inside eye masks.

Do not move the eyebrows during a normal blink.
