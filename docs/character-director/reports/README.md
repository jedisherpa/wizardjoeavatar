# Independent Specialist Reports

Phase 0 reports are kept separate from synthesis so source findings remain
auditable. Each report must contain evidence, exact files and symbols, findings,
risks, recommended changes, rejected alternatives, and verification
requirements.

All eleven independent reports are frozen in this directory. The single
synthesis round is recorded in `../PHASE0_SYNTHESIS.md`; it resolves conflicts
without silently rewriting the underlying reports. The accountable execution
plan is `../IMPLEMENTATION_WORKFLOW.md`.
