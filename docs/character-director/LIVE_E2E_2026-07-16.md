## Connector Correction

A real governed turn exposed an acknowledgement identity mismatch when a
loading speech source correctly did not preempt visible main media. The Python
coordinator had paired the accepted speech sequence with the visible main
media epoch. `MediaSessionCoordinator` now tracks the last accepted incoming
media epoch independently from the last visible snapshot and returns that
single-source cursor in its acknowledgement.

The regression test uses visible main epoch 3 and incoming speech epoch 4. It
asserts that the acknowledgement accepts sequence 1 at epoch 4 while main epoch
3 remains visible. The focused media-session suite passed 38 tests and the full
Python suite passed 428 tests after the correction.
