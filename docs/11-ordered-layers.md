# Build the Character from Ordered Layers

Create these layers:

```text
1. rear staff
2. rear arm
3. rear robe
4. boots
5. robe base
6. inner magenta robe
7. gold trim
8. belt and buckle
9. front arm
10. hand
11. neck
12. head
13. ears
14. beard
15. mouth
16. nose
17. eyes
18. eyebrows
19. hat brim
20. hat crown
21. hat stars
22. front staff
23. cyan orb
24. magic effects
```

Every layer must have:

- stable ID
- z-index
- cell mask
- palette entries
- glyph entries
- local origin
- anchor references
- visible views
- optional animation channel

Do not flatten these layers at authoring time.
