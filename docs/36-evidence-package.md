# Evidence Package

Produce:

```text
evidence/wizard/
├── repository-audit/
├── palette/
├── masks/
├── views/
├── expressions/
├── walk-cycles/
├── environment/
├── golden-images/
├── visual-diffs/
├── codec-vectors/
├── movement-traces/
├── circle-traces/
├── performance/
├── browser/
├── test-results/
├── demo/
└── FINAL_VERIFICATION.md
```

`FINAL_VERIFICATION.md` must include:

- commit SHA
- source reference image path
- exact local run command
- exact test command
- production build command
- all test totals
- failed tests
- skipped tests
- visual evidence
- movement evidence
- codec evidence
- FPS measurements
- bandwidth measurements
- remaining limitations
