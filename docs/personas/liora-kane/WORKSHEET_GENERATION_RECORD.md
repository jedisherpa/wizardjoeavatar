## Production boundary

These PNGs are approved canonical pose references, not runtime sprites. Runtime implementation must extract approved panels into aligned deterministic direct square-cell data with at least a four-cell safety inset. Runtime code must never decode or display the worksheet PNGs. Pose-local pivots, roots, baselines, prop attachments and transition timing still require implementation and live testing in the later runtime phase.
