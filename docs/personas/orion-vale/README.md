# Orion Vale — Character Direction

- School function: Wisdom / Inquiry
- Source assignment: long-haired scholar with question journal
- Canonical art: `canonical-voxel.png`

## Identity lock

Immutable: long wavy brown hair; broad forehead; round gray glasses; blue-gray eyes; warm brown beard; dark-brown coat; cream shirt; red-and-cream checked tie; brown question journal with gold question mark.

Performance: reflective, patient, quietly playful, and anti-dogmatic. Use unhurried journal holds, small question-offering gestures, thoughtful gaze shifts, distinction-drawing hands, and pauses that leave room for the user.

Signature actions: find the hidden question, isolate an overloaded word, offer a testable handle, name where a lens breaks, Socratic listening, and put inquiry down during distress.
