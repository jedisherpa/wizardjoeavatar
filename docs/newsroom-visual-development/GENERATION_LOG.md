# Generation Log and Limitations

## Method

All eight full-size references were created with the built-in OpenAI image-generation tool using the two supplied Wizard Joe images as style references. The common prompt contract required: 16:9 framing; crisp discrete square cells; bright white fields; cobalt, cyan, gold, magenta, brown, and white; readability at 320x180; no photorealism, gradients, dark atmosphere, text, logos, or watermarks. Set prompts explicitly prohibited characters. Camera-board prompts explicitly limited characters to composition reference.

The set-specific prompt set was:

- Main anchor: centered cobalt/cyan desk, gold edge, blank central and side displays, practical light columns, clear actor zone.
- Explainer: large blank wall camera-right, open presenter zone camera-left, stage marks, parked lectern, practical lights.
- Interview: two chairs, low table, blank rear display, two actor zones, open side paths.
- Magical field: blank portal frame, field lectern, platform, ring devices, separable sparkle clusters.
- Studio props: desk front/side, chair front/side, large/small display, lectern, practical light.
- Broadcast/magic props: blank source card, lower third, correction badge, two-box, magic ring, portal console, sparkle cluster, beacon.
- Board A: establishing wide, anchor medium, close-up, over-shoulder graphic, two-shot.
- Board B: profile, high angle, low angle, dolly start, dolly end, cutaway inserts.

## Generation limitations

1. The broadcast/magic sheet rendered two ring-like magical devices and did not produce a distinct correction badge. Treat the extra ring as an alternate device; author the correction badge directly as a cell graph or generate a dedicated follow-up reference.
2. Board B's low-angle field panel contains a pale blue exterior/sky patch and soft depth cues. These violate the final flat-white, no-gradient style and must not be traced.
3. Some full-size references contain generated bevels, gloss, soft shadows, perspective floor lines, or color variation inside cells. These are image-model artifacts. The 320x180 graph must use flat approved palette colors.
4. Board A and Board B vary Joe's face, beard, hands, robe, and staff. None of those character pixels are canonical. Use only approved Rust pose graphs.
5. Object sheets are not guaranteed orthographically exact or mutually scale-consistent. Normalize dimensions, anchors, and cell size during graph authoring.
6. The forced 320x180 previews are review aids produced by raster reduction. They are not valid sources for automatic cell extraction.

## Approval boundary

This visual-development pass approves composition directions for further tracing and review. It does not approve any generated raster, traced graph, newsroom scene, or camera transition for production runtime use.
