# Implement Expressions

Create these expression overlays:

```text
neutral
happy
thinking
surprised
worried
amused
confident
focused
skeptical
explaining
```

Each expression must define:

- left eyebrow
- right eyebrow
- left eye
- right eye
- mouth
- optional head offset
- optional beard or cheek highlight

Expressions must alter only the face and approved nearby cells.

They must not redraw the whole character.
