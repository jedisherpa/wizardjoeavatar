# Create an Automated Full Demonstration

Create:

```text
tools/demo_wizard_avatar.py
```

The demonstration must perform this exact sequence:

1. Load the fixed white environment.
2. Spawn the wizard in the center.
3. Idle for three seconds.
4. Blink.
5. Change to happy.
6. Speak one test line.
7. Walk left.
8. Turn and walk right.
9. Walk away from the camera.
10. Walk toward the camera.
11. Walk front-left.
12. Walk back-right.
13. Walk one clockwise circle.
14. Walk one counterclockwise circle.
15. Walk a figure-eight.
16. Stop in the center.
17. Think.
18. Point.
19. Explain.
20. Cast magic.
21. React.
22. Return to neutral idle.
23. Continue idling for ten seconds.

The demo must run through the real ASCILINE WebSocket path in a browser.
