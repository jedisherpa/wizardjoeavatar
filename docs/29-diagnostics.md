# Add Diagnostics

Display a development panel with:

- world x
- world z
- screen x
- screen y
- display scale
- facing direction
- velocity
- target point
- walk phase
- current action
- current expression
- mouth state
- FPS
- frame sequence
- codec tag
- raw frame size
- encoded frame size
- delta cell count
- keyframe count
- dropped frame count
- reconnect count

Do not expose secrets.
